# create-pr

Reusable skill for creating or updating pull requests with purpose-first context, scoped change summaries, and verification evidence.

## Triggering

Invoke explicitly:

```text
$create-pr
```

## What It Covers

- Gather branch/base/diff context
- Detect existing PR vs new PR creation
- Write reviewer-focused PR body structure
- Apply PR edits via `gh`
- Validate PR body quality and report outcome

## Compatibility

- Codex: Yes
- Claude Code: Yes
