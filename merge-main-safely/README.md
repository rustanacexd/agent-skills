# merge-main-safely

Reusable skill for safely merging the latest upstream base branch into the current branch with conservative conflict handling.

## Triggering

Invoke explicitly:

```text
$merge-main-safely
```

## What It Covers

- Branch safety and clean-tree checks
- Base branch resolution and fetch
- Conflict-aware merge strategy
- Post-merge validation with repo-native checks
- Merge outcome reporting

## Compatibility

- Codex: Yes
- Claude Code: Yes
