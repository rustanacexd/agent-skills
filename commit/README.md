# commit

Reusable skill for creating focused, reviewable conventional commits with safe staging discipline.

## Triggering

Invoke explicitly:

```text
$commit
```

## What It Covers

- Partition working-tree changes by intent
- Stage one logical unit at a time
- Write conventional commit messages
- Verify each commit result and report remaining changes

## Compatibility

- Codex: Yes
- Claude Code: Yes
